# Skill Files

This directory contains skill files for the Plektis protocol. Some are functional domain skills, others are format examples.

## Included Skills

| Skill | Type | Purpose |
|---|---|---|
| [business-architecture-foundations](business-architecture-foundations/SKILL.md) | Domain Intelligence | Foundational BA concepts: capability mapping, value streams, BMM, PPIT. A working starting point for practitioners. |
| [example-domain-intelligence](example-domain-intelligence/SKILL.md) | Domain Intelligence | Format example showing the required structure for domain intelligence skills. |
| [example-operational-playbook](example-operational-playbook/SKILL.md) | Operational Playbook | Format example showing the required structure for operational playbooks. |

## Building Your Own

1. Start with `business-architecture-foundations` as a reference for what a real skill file looks like
2. Copy the relevant template from `sdk/templates/` for your skill type
3. Fill in your domain expertise, workflows, and quality standards
4. Follow the format specification at `reference-model/L1/6.4-skill-file-architecture.md`

Skill files are AAIF-conformant (agentskills.io). They work with any compatible agent.

## Foundations Approach

The `business-architecture-foundations` skill encodes publicly available concepts from BIZBOK, BMM, and established EA practice. It provides the vocabulary and structural patterns for business architecture work.

Practitioners extend it with their own sector-specific knowledge, assessment instruments, engagement methodologies, and stakeholder calibration patterns. The foundations are shared. The application is where value is created.
