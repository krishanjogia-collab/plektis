# Skill File Templates

Starter templates for creating Plektis skill files. Copy one, fill in your domain expertise, and follow the format specification.

| Template | Purpose |
|---|---|
| [Domain Intelligence](TEMPLATE_DomainIntelligence.md) | For encoding persistent domain context (what the agent knows) |
| [Operational Playbook](TEMPLATE_OperationalPlaybook.md) | For encoding executable workflows (what the agent does) |

For the full format specification, see [Skill File Architecture](../../reference-model/L1/6.4-skill-file-architecture.md).
