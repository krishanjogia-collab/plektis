# SDK

## Orientation
Skill files are structured `.md` files that encode practitioner expertise as machine-readable instructions. 

There are two primary types of skill files:
1. **Domain Intelligence (Persistent Context):** Represents deep institutional, environmental, or strategic knowledge regarding a domain.
2. **Operational Playbook (Executable Workflows):** Represents a series of step-by-step methodologies or procedural tasks that drive an engagement.

These skill files are fully conformant with the **AAIF Agent Skills standard** (agentskills.io).

## Usage
To use the templates, copy an existing template from the `templates/` directory, fill in the customized sections for your domain or operational workflow, and deploy it to your environment.
