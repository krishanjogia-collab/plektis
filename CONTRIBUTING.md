# Contributing

This project is currently maintained by a single author.

**Forking and building on Plektis is welcome and encouraged.** The protocol is
released under CC BY-SA 4.0 - you are free to use it, adapt it, and build your
own configurations on top of it, with attribution and share-alike.

**Formal contributions are not yet being accepted.** A formal contribution model
(pull requests, governance, review process) will be established as the community
grows. For now, if you find this useful or want to build on it, open an issue to
start a conversation.
