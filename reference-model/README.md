# Reference Model

## Orientation
- **Level 0 (The Node):** Covers the node as a system, the SenRef cognitive cycle, the interaction layer, escalation triggers, and the persistence model.
- **Level 1 (Subsystems):** Covers specific subsystems including Driver Capability, Machine Capability, Interaction Protocols, Skill File Architecture, Application Mounting Protocol, and Multi-Agent Coordination.

**Current Status:**
- Level 0 is drafted and ready for peer review.
- Level 1 Sections 6.1, 6.2, 6.3, 6.4, and 6.6 are stable. Sections 6.5 (Application Mounting Protocol) and 6.7 (Ecosystem Integration Protocol) are early drafts. Every part of the reference model is at v0.1.0 for the initial public release.
