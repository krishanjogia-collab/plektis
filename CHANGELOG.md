# Changelog

All notable changes to Plektis are documented in this file.

## [0.1.0] - 2026-06-02

Initial public release. Plektis is an open protocol for human-AI teaming. This release includes:

- **Level 0 reference model:** the node, the SenRef cognitive cycle, the interaction layer, the persistence model, and the escalation triggers.
- **Level 1 specification (Sections 6.1 to 6.7):** driver and machine capability, the interaction protocol, the skill file architecture, application mounting, multi-agent coordination, and ecosystem integration.
- **SDK:** the skill file format and templates, with the Business Architecture Foundations skill.
- **Deployment Profiles:** Business Architecture, and Portfolio and Transaction Strategy.
- **Integration Profiles:** Anthropic Managed Agents, Google Vertex AI Agent Engine, and AWS Bedrock AgentCore.
